项目内安装
npm install webpack --save-dev //安装
npm uninstall webpack --save-dev //卸载

全局安装
npm install webpack -g //安装
npm uninstall webpack -g //卸载

指定版本号安装 webpack

npm install webpack@1.14.1 -g
npm install webpack-dev-server@1.14.1 -g
npm install gulp@3.9.1 -g
npm install bower@1.8.0 -g
